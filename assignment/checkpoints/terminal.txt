# 8 - Why would you run the command 'git status'?
	To view details about the condition of your working directory.  
	It will show the branch you are on and what files have been modified since your last commit.  
	It will also show you what is in the staging area and whether or not certain changes are ready to be committed.

# 9 - What does the command 'git add .' do?
	Adds all files in the current directory to the staging area, in prepartion to be committed.

# 10 - What is the command to push your committed changes to your repo on GitHub?
	git push -u origin master
